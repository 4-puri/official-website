DOWNLOAD FLASH BLOCKCHAIN SNAPSHOT

In which cases you need to download Flashcoin blockchain?

  • When you need to get full Flashcoin node
  • When you installing wallet like Flashcoin Core to new device
  • If your blockchain was crashed and you need get it back to normal fast

Step 1: Download flashcoin_blockchain_snapshot file:

Download
(https://flashcoin.io/downloads/flashcoin_blockchain_snapshot.02Apr2019.tar.gz)
(SHA256: b6ec021a900d54379006120cf1bea366e491dc08a1bf41f486cb91d631f4f67e)

(SHA512: 143295dae755988ada6b819ddfd566514fc84cf480317c38a7ad9782bc861ca4080d3a4f731041a5e7c701fa84e820045fc838b4b78baf0db9e52ac0de197e64)

The blockchain file has size 1.1GB.

Step 2: Unzip the flashcoin_blockchain_snapshot_*.tar.gz contain:

The file contain:
.\blocks
.\chainstate
.\peer.dat

Step 3: Copy all the content to the Flashcoin data folder on your local machine. The default location:

1) Windows OS:

By default Flashcoin will put its data here:
C:\Documents and Settings\YourUserName\Application data\Flashcoin (XP)
C:\Users\YourUserName\Appdata\Roaming\Flashcoin (Vista and 7)

2) Mac OSX:

By default Flashcoin will put its data here:
~/Library/Application Support/flashcoin/

3) Unix*:

By default Flashcoin will put its data here:
~/.flashcoin/

Step 4: Start your Flashcoin QT-Wallet

It can take some time to rescan (reindex) all the blockchain data.
